Pac-Man Game - Group 16

Java-based implementation of the classic Pac-Man arcade game.

Overview

Collaborated with a team to develop a Java-based implementation of the classic Pac-Man arcade game. Integrated JavaFX for the GUI, implemented game mechanics, and developed features like leaderboards and dynamic difficulty levels.

Key Features

  • Interactive GUI developed using JavaFX.
  • Dynamic difficulty levels that increase with each round.
  • Implemented leaderboards to track player scores.
  • Distinct ghost behaviors with varying speeds and strategies.
  • Background music and sound effects for an immersive experience.

Installation & Setup

To set up the Pac-Man Game locally, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/shazilfkhan/pacman-game.git
  2. Navigate to the project directory: 
    cd pacman-game
  3. Ensure JavaFX SDK is installed and configured on your system.
  4. Import the project into your preferred IDE (e.g., IntelliJ IDEA).
  5. Run the main class to start the game.
