Pac-Man Game - Group 16

Java-based implementation of the classic Pac-Man arcade game.

Project Overview

A modern spin on the classic Pac-Man arcade game, built in Java for the CPSC 219 course at the University of Calgary. The game features an interactive GUI, ghost AI, dynamic difficulty, and a leaderboard system.

The primary objective is to collect pellets and fruits while evading ghosts. Each new level intensifies the challenge by increasing the number of ghosts and their movement speed.

My Role & Key Contributions

  • Implemented Ghost AI: Developed logic to handle distinct ghost behaviors, including different speeds and vulnerability states after Pac-Man eats a fruit.
  • Integrated JavaFX UI: Built the main game window, menus, and scoreboard layout, ensuring a user-friendly and responsive interface.
  • Leaderboards & Persistence: Created the database/file handling for storing player scores and nicknames, enabling the leaderboard feature.
  • Collaborative Version Control: Used GitHub to manage code merges and resolve conflicts, ensuring smooth teamwork with all group members.

Challenges & Learning Outcomes

  • Challenge: Ensuring seamless collision detection between Pac-Man and ghosts in a constantly updating GUI.
    Solution: Implemented a well-defined game loop using JavaFX’s animation timeline, updating positions and detecting collisions each frame.
  • Challenge: Balancing difficulty curves so that each new level feels progressively harder but remains enjoyable.
    Solution: Dynamically increased ghost speed and number of ghosts. Also conducted user tests to refine difficulty algorithms.
  • Takeaway: Strengthened my understanding of JavaFX, concurrency in Java, and collaborative development using version control systems.

Gameplay & Instructions

Game Objective

  • Moving Pac-Man: Use arrow keys to navigate the maze.
  • Eat Pellets & Fruits: Clear the maze by collecting them all.
  • Avoid Ghosts: Losing all three lives ends the game.
  • Vulnerable Ghosts: After eating fruit, Pac-Man can temporarily eat ghosts for bonus points.

Rules

  • Pac-Man starts with three lives.
  • The game ends once all lives are lost.
  • The number and speed of ghosts increase each level.
  • Ghost behaviors change when they become vulnerable.

How to Run the Game

IDE: Recommended to import the project in IntelliJ IDEA or Eclipse. Java: Use Java 22/23 for compatibility. JavaFX SDK: Make sure JavaFX libraries are properly configured.

After setup, run the main class to start the game and enjoy Pac-Man!
